Built for the teacher who needed it

My wife Criste has been teaching for over twenty years. Every morning she'd juggle five different apps, three browser tabs, and a paper binder just to plan one day of instruction. When she woke up sick at 5am, creating substitute plans meant an hour of frantic typing before the school day started. The tools available were either designed for administrators, priced for districts, or built by people who'd never stood in front of a classroom.

So I built her one. Not a prototype ‚Äî a real platform. AI-powered lesson plans in 90 seconds. Emergency sub plans ready before the bus arrives. Parent communication templates, standards alignment, behavior management tools, and an entire emergency toolkit with 12 categories of solutions for everything that can go wrong in a school day.

She uses it every single day. Not because she has to ‚Äî because it actually saves her time. That's the only metric that matters for teacher tools: does it give time back?

My Teachers Toolbox isn't theoretical. It's been tested in a real classroom, refined through daily use, and built around the workflows that actual K-5 teachers follow. Every feature exists because a real teacher asked for it.

Architecture
How It's Built
Frontend
Next.js 15 with React 19, Server Components, and Tailwind CSS. 71 components across 16 module directories ‚Äî planner, communication, documents, emergency, standards, activities, AI, and more.
Backend
99 API routes handling lesson generation, communication, document creation, export, Stripe billing, user management, standards search, activities, and the complete emergency toolkit.
AI Layer
DeepSeek integration with 5 custom prompt systems ‚Äî lesson generation, week planning, newsletter writing, runsheet creation, and reflection coaching. Embeddings for semantic search. Caching layer for cost control.
Database
Supabase PostgreSQL with tables for profiles, lesson plans, activities, standards, favorites, API usage tracking, and billing. Row Level Security for multi-tenant data isolation.
Export & Documents
PDF generation via jsPDF and pdf-lib. DOCX generation via the docx package. Lesson plan exporter with formatting, name label generator, and curriculum document builder.
Production Infrastructure
Vercel deployment with Sentry error monitoring and Vercel Analytics. Stripe webhook handling for subscription management. Zod validation on all API inputs. SEO-optimized blog with 3 content pages.
